Tipos de utilização das CSS

Temos 3 formas de incluir um estilo CSS em seu documento HTML, são eles:


Inline:

Quando o estilo é direcionado expecificamente para a tag que você deseja estilizar

Interno
É possível incluir o estilo em tudo que você selecionar no próprio documento HTML, sendo necessário incluir uma tag <style> dentro da área de Cabeça <head>
Externo:
Basicamente é criar um documento a parte para fazer a estilização do documento HTML

Adicionando Cores

Você pode abrir uma outra página para usar somente CSS: Não esqueça de colocar o @charset "UTF-8" para não ter incompatibilidade com acentos dentre outros.

Seletores:

* (asterisco) é um seletor de configuração global das CSS.
Crie o seletor da tag que você deseja adicionar estilo. body, h1, h2, p... sempre adicionar o {} após o seletor.

Cor da letra e plano de fundo:

Color: ; muda a cor da letra

background-color:; muda a cor de fundo

background-image: linear-gradient ( )Você adiciona um Degradê ao fundo da página exemplo: 

         
    body {
         background-image: linear-gradient(to top, black, grey, white)
    }
vai fazer um degradê do preto para o branco ao invés de utilizar to top, right, left, bottom... Pode utilizar em graus, indicando o numero e digitar "deg" em seguida.

Ao invés de linear-gradient, pode utilizar radial-gradient apos os decimais das cores, você pode incluir a porcentagem da cor que vai aparecer.

Configuração de Caixas

border-radius: Irá arredondar as pontas de uma caixa de texto expecificada. Utilizar em pixels 10px, por exemplo

box-shadow: Irá criar sombra nas letras.
Parâmetro 1: o quanto a sombra irá para o lado
Parâmetro 2: o quanto a sombra irá para baixo
Parâmetro 3: o quanto a sombra irá espalhar
Parâmetro 4: a cor que a sombra terá

Configurações de texto:

text-align: alinhará o texto da forma que você quiser

text-shadow: irá incluir sombra no texto selecionado mesmos parâmetros do box-shadow
