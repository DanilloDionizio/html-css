Como o computador entende?

Um computador não entende nossa linguagem, apenas em Binário, que são Zeros (0) e Ums (1), também representado como Ondas Quadradas, por isso foram feitos programas que "traduzem" nossa linguagem para a linguagem do computador, facilitando a comunicação com o mesmo. (0) ou (1) é chamado de Bit, que vem de Binary Digit.

Como funciona o armazenamento do Computador?

Conforme o texto acima, 0 e 1 = 1Bit, a porção minima para representar um dado é de 8bits ou 1byte.

Como o computador calcula isso?

Computadores trabalham com base 2 de referência, sendo sempre 2 10.

Quais os múltiplos de byte?

Múltiplos de Byte são:
1024Bytes = 1KiloByte
1024KiloBytes = 1MegaByte
1024MegaBytes = 1GigaByte
1024GigaBytes = 1TeraByte
1024TeraBytes = 1Petabyte
1024PetaBytes = 1HexaByte
1024HexaBytes = 1ZetaByte
1024Zetabytes = 1YotaByte
dentre outros múltiplos....

Imagens

O que é uma imagem?

Uma imagem é um conjunto de pontos que vai desde o ponto 00, que vai contando desde o ponto de coluna até o ponto de linha, fazendo a multiplicação dos mesmos.

formatos de imagens:

JPEG

Quanto maior o arquivo, mais dificil é para carrega-lo e para coloca-lo na internet, seguindo este princípio, sabemos que a internet não pode ter arquivos pesados, é necessário compactar os mesmos. O formato Jpeg é o melhor formato de compactação de imagem para incluir imagens grandes, com boa qualidade e com pouco consumo de memória.

PNG

Este formato foi criado pela W3C (world wide web consortion).
Basicamente o formato PNG foi criado para substituir o formato GIF, sendo um formato bem antigo mas a utilização do mesmo voltou com tudo após as redes sociais.
Mesmo ele sendo um formato com imagem horrível, permite transparência e animação, já o PNG não permite animação, mas permite transparência.
O PNG também é compactado, não tanto quanto o JPG, mas este formato também compacta os arquivos e permite transparência.

Big e Small

A tag <big>, segundo a W3C ficou Obsoleta, mas a <small> não, porquê?
A tag <big> foi obsoleta exatamente porque não há um significado, não há semântica, se for necessário incluir um tamanho maior à frase é considerado forma e pode ser alterado nas CSS's, tudo trata-se de semântica.
Já a <small> tem um significado não expecificado pela W3C, mas, segundo o professor Gustavo Guanabara:

Um exemplo da semântica do <small> é tipo em um contrato, que às vezes está descrito: Leia as letras miúdas. que é geralmente definido como observações importantes.

Através do <small> você ainda consegue dar sentido ao que está sendo escrito. Já no <big> não, por isso esta já ficou obsoleta.

Deletado e Inserido

A tag <del> define um(a) letra, número, palavra, frase ou texto que pode ser lido, porém, deve ser desconsiderado durante a leitura.
A tag <ins> define um(a) letra, número, palavra, frase ou texto que substitui o que foi incluido no <del> e, esse sim, deve ser considerado durante a leitura.

Sobrescrever e Subescrever

A tag <sup> nada mais é que colocar o que desejar acima do que foi escrito anteriormente.
A tag <sub> é colocar o que desejar abaixo do que foi escrito anteriormente.
Por exemplo:
1510 < aqui foi utilizada a tag <sup>
1510 < aqui foi utilizada a tag <sub>

Tipos de listas

Dentro das Ordered List e Unordered List devem ser incluidas a tag <li> List Item para colocar um item na lista

Lista Ordenada / Ordered List

A Lista Ordenada <ol> significa uma lista organizada por sequências de números ou letras.
Utilizada geralmente para definir ingredientes, algo que precisa de um passo a passo.
exemplo:

Receita de bolo:

  1. 2 ovos
  2. 300g farinha de trigo
  3. 1colher sopa fermento
Ela vem por padrão a sequência numérica, mas você pode mudar incluindo dentro da tag o parâmetro type.
Por exemplo:
<ol type="a"> para sequência alfabética minúscula, altere dentro das aspas para A (maiúsculo) e terá sequência alfabética maiúscula.
Altere para i minúsculo, e terá sequência de numeração romana minúscula.
Altere para I maiúsculo e terá sequência de numeração romana maiúscula.

Lista não ordenada / Unordered List

A lista não ordenada <ul> significa uma lista que não necessita de organização, como por exemplo:
Lista de compras:
  • Arroz
  • Feijão
  • Ovos
  • Papel higiênico
Diferente da Ordered List, esta não é classificada por sequências, mas também há variações type como a <ol>
<ul type="disc"> vem por padrão.
Altere dentro das aspas para "square" e terá um quadradinho como marcador.
Altere para "circle" e terá um circulo incolorido.
Demonstrações:
  • Esse vem por padrão
  • Quadradinho
  • Circulo traçado
